ejercicio nro 1

Tengo desarrollada la siguiente función en javascript
function suma (a,b,callback){
let c = a +b;
callback();
} ¿Que es el tercer parámetro recibido? ¿En que casos es obligatorio desarrollar este tipo de funciones? Realizar un llamado a la función de ejemplo.

Respuesta

La función suma toma dos parámetros a y b y un tercer parámetro callback. El tercer parámetro, callback, es una función que se pasa como argumento y se ejecutará al final de la función suma.

Este patrón se llama "callback". Los callbacks son funciones que se pasan como argumentos a otras funciones y se ejecutan después de que alguna operación ha sido completada. En este caso, la función callback se ejecuta después de calcular la suma de a y b.

Este tipo de funciones con callbacks son útiles en situaciones asíncronas o cuando queremos que cierto código se ejecute después de que una operación particular haya concluido.
